# Echo Telegram Bot

Foydalanuvchiga o'zi yozgan xabarlarni interaktiv qaytaruvchi bot.

## Muammo

Ushu bot orqali ayni damda mavjud muammoni emas balki telegram bot qanday ishlashini o'rganish va telegram bot lar olamiga kirishdagi boshlang'ich qiyinchiliklarni yengillashtirish

## Maqsad

- Telegram bot va Telegram API qanday ishlanishi tushinish
- Foydalanuvchi ning telegramdagi bazaviy ma'lumotlari bilan ishlash
- Dotnet da String lar ustida amaliyot o'tkazish
- Dotnet da Regex ni o'rganish
- Interaktiv bot tayyorlash
- File lar bilan ishlashni o'rganish

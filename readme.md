# Murod Echo Bot - Milestone 1

Bu loyiha Milestone 1 talablari asosida yaratilgan oddiy Echo bot hisoblanadi.

###  O'rnatish Setup
1. Repozitoriyani kompyuteringizga yuklab oling (Clone).
2. Loyihani JetBrains Rider yoki Visual Studio dasturida oching.
3. Telegram.Bot NuGet paketini o'rnating.
4. Program.cs faylidagi Token o'zgaruvchisiga o'z botingiz tokenini yozing.

###  Foydalanish Usage
1. Dasturni ishga tushiring (Run).
2. Telegram orqali botga xohlagan matningizni yozing.
3. Bot sizga nima [matn] yozsangiz ushani qaytaradi.

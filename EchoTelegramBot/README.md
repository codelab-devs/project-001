# EchoTelegramBot
# Setup
- Repository linki copy qilinib yangi papka terminalida 'git clone' buyrug'i orqali clone qilib olinadi.
- Shaxsiy bot token Program.cs faylidagi tokenga joylashtiriladi va botga TelegramBot nuget paketini o'rnating.
# Usage
- Dastur 'dotnet run' komandasi orqali ishga tushiriladi.

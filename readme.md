# Murod Echo Bot - Milestone 1

Bu loyiha Milestone 1 talablari asosida yaratilgan oddiy Echo bot hisoblanadi.

## Muammo
Ushbu bot orqali ayni damda mavjud muammoni emas, balki Telegram bot qanday ishlashini o'rganish va botlar olamiga kirishdagi boshlang'ich qiyinchiliklarni yengillashtirish maqsad qilingan.

## Maqsad
Telegram bot va Telegram API qanday ishlashini tushunish
- Foydalanuvchining telegramdagi bazaviy ma'lumotlari bilan ishlash
- Dotnet'da String'lar ustida amaliyot o'tkazish
- Dotnet'da Regex'ni o'rganish
- Interaktiv bot tayyorlash
- Fayllar bilan ishlashni o'rganish

## Setup
1. Repozitoriyani kompyuteringizga yuklab oling (Clone).
2. Loyihani JetBrains Rider yoki Visual Studio dasturida oching.
3. `Telegram.Bot` NuGet paketini o'rnating.
4. `Program.cs` faylidagi `Token` o'zgaruvchisiga o'z botingiz tokenini yozing.

## Usage
1. Dasturni ishga tushiring (Run).
2. Telegram orqali botga xohlagan matningizni yozing.
3. Bot sizga yozgan matningizni qaytarib yuboradi.
